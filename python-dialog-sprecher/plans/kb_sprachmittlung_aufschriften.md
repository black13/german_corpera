# K173-K176: Sprachmittlung — Aufschriften

Related core planning note: `kb_visible_text_carriers.md`.

## Core distinction

K174-K176 = geschriebene Aufschriften, nicht gesprochene Durchsagen.

**sehen → lesen → weitergeben**
Not: hören → verstehen → weitergeben.

K174-K176 betreffen sichtbare Schrift im Raum.

## The three scenes

| KB   | Ort                        | Aufschrift        | Übersetzung       |
|------|----------------------------|-------------------|-------------------|
| K174 | Geschäftstür               | z.B. "fermé"      | "geschlossen"     |
| K175 | Toilettentür im Restaurant | z.B. "caballeros"  | "Herren"          |
| K176 | Hotel-Informationstafel    | z.B. "breakfast"   | "Frühstück"       |

## Parent skill

K173 (detailed): Kann einfache anderssprachige Informationen von Schildern und Aufschriften Deutschsprachigen in Einzelwörtern auf Deutsch weitergeben.

## Speech act

"Das bedeutet..." / "Das heißt..."

- "Was steht da?" → "Da steht 'fermé'. Das bedeutet 'geschlossen'."
- "Was heißt 'breakfast'?" → "Das bedeutet Frühstück."

## Why these are the easiest Sprachmittlung

The source text doesn't disappear. A spoken sentence is gone after you hear it. An Aufschrift stays on the door. You can look at it, think, and then say "Das bedeutet..."

## Word breakdown

- **Aufschrift** = auf (on) + Schrift (writing) = inscription, label
- **Geschäftstür** = Geschäft (shop/business) + Tür (door)
- **Informationstafel** = Information + Tafel (board)

## Der Gegenstand bleibt — why signs are the easiest input

The Aufschrift is a Gegenstand that waits. It doesn't move, it doesn't
disappear, it doesn't speak faster than you can think. This is the key
difference between written and spoken Sprachmittlung:

- **Schild an der Tür**: bleibt. You can look, think, look again.
- **Tafel im Hotel**: bleibt. You can point at it.
- **Speisekarte**: bleibt. You can read it twice.
- **Gesprochene Frage**: weg. You heard it once, now produce.

The whole K173-K176 cluster depends on this: the Gegenstand is still
there while you formulate your German sentence.

## Full Sprachmittlung map (K153-K176)

Two directions x two input channels:

```
                        INPUT
                 Spoken          Written
              ┌────────────┬────────────────┐
  DE → other  │ K155-K162  │  K163-K166     │
  language    │ hear DE,   │  read DE text, │
              │ relay in   │  relay in      │
              │ shared lang│  shared lang   │
              ├────────────┼────────────────┤
  Other → DE  │ K169-K172  │  K173-K176     │
              │ hear other │  see other     │
              │ lang, say  │  lang sign,    │
              │ in DE      │  say in DE     │
              └────────────┴────────────────┘
```

Difficulty (easiest → hardest):
1. K173-K176: read foreign sign → say in DE (text stays)
2. K163-K166: read DE text → relay in shared lang (text stays)
3. K169-K172: hear other lang → say in DE (input gone)
4. K155-K162: hear DE → relay in shared lang (input gone, catch numbers)

## K175 deep dive: Toilettentür im Restaurant

KB: Kann in einem Restaurant die Aufschrift (z. B. „caballeros") auf
der Toilettentür für einen deutschsprachigen Touristen übersetzen.

### Scene ladder

- Big scene: Restaurant in a foreign country
- Small scene: hallway, two doors, foreign labels
- Roles: learner as mediator, German tourist who can't read the sign
- Speech act: translate the door label into German
- Utterance: "Das bedeutet Herren." / "Das ist die Damentoilette."

### Good offshoots for K175

- "caballeros" → "Herren"
- "damas" / "señoras" → "Damen"
- "WC" → "Toilette" (international but still needs explaining sometimes)
- "toilettes" → "Toiletten"
- "sortie" → "Ausgang"
- "entrée" → "Eingang"
- "privé" / "staff only" → "Nur für Personal"
- "occupé" → "besetzt"
- "libre" → "frei"

### Bad offshoots for K175

- Ordering food (that's K161, spoken mediation)
- Reading the menu (that's K092, Rezeption schriftlich)
- General restaurant conversation
- Discussing bathroom vocabulary beyond the sign

### What the Gegenstand looks like

The student should picture: a door. On the door is one word. A German
person points at it and says "Was bedeutet das?" The word stays on the
door while the student thinks and answers.

This is the same pattern as K174 (Geschäftstür) and K176
(Informationstafel). Only the building and the word change. The
cognitive task is identical: read → translate → say.

### Grammar tools

- "Das bedeutet..." / "Das heißt..."
- "Das ist die/der..." (with article if naming the room)
- No complex grammar needed — single words or minimal phrases
