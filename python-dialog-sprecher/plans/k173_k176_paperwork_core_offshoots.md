# K173-K176 Paper Work: Core KB And Offshoots

This note captures the paper-planning idea behind the classroom generator:

> big scene -> small scene -> speech act -> text/utterance -> lexical field -> grammar

The teacher should not begin with grammar as an abstract topic. The teacher should begin by naming the current Kannbeschreibung as the work goal, then show which grammar and vocabulary tools help that goal.

## Core Family

K173 is the broad pattern:

- K173: give simple foreign-language information from signs/inscriptions to German speakers in German single words.

K174-K176 are concrete scene offshoots of that pattern:

- K174: shop door sign for a German-speaking tourist.
- K175: restaurant/toilet-door inscription for a German-speaking tourist.
- K176: hotel information-board term for a German-speaking guest.

The important relationship is not only vocabulary. It is role plus action:

- Reader sees a short written sign.
- German-speaking person does not understand the sign.
- Student acts as mediator.
- Student gives the meaning in simple German.
- Student checks or repairs understanding if needed.

## K176 As A Core Day

Current KB:

> Kann im Hotel einfache Begriffe auf der Informationstafel (z. B. „breakfast“) für einen deutschsprachigen Gast übersetzen.

For this day, K176 is the core. K173, K174, and K175 are siblings/offshoots that can help planning, but they should not take over the lesson.

Good offshoots for K176:

- `breakfast` -> `Frühstück`
- `check-out` -> `Zimmer bis elf Uhr verlassen`
- `luggage storage` -> `Gepäckaufbewahrung` / `Koffer aufbewahren`
- `housekeeping` -> `Zimmer sauber machen`
- `parking` -> `Parkplatz`
- `reception` -> `Empfang` / `Schlüssel holen`
- `do not disturb` -> `Bitte nicht stören`

Bad offshoots for K176:

- broad talk about why people go to hotels
- personal data registration unless the KB specifically asks for registration
- general hotel roleplay that does not require translating/explaining a board term
- end-of-class praise when the student has not performed the target act

## Lesson Shape

The teacher opening should look like:

> Heute üben wir: Ein Gast versteht ein Wort auf der Informationstafel im Hotel nicht. Sie erklären das Wort auf Deutsch. Wir benutzen Sätze wie: "Das bedeutet..." und Zeitwörter wie "von...bis" oder "bis elf Uhr."

Then immediately create the small scene:

> Ein Gast zeigt auf "breakfast". Was bedeutet das?

## Grader Shape

The grader should judge four things separately:

- Teacher goal alignment: did the teacher prompt force the current KB?
- Task alignment: did the student attempt mediation/translation?
- Communicative success: would a patient German speaker understand?
- A1 fit: was broken but functional A1 German accepted?

This matters because a student can drift if the teacher prompt drifts. In the latest K176 run, round 5 exposed this correctly: the teacher praised or discussed loud speaking instead of giving the student a concrete K176 task, and the grader marked `NEEDS REDIRECT`.

## Paper Work Use

For each KB cluster, write this before running the conversation:

- Core KB for the day.
- Sibling/offshoot KBs.
- Big scene.
- Small scene.
- Required speech act.
- Acceptable utterance forms.
- Vocabulary field.
- Grammar tools that make the speech act work.
- Things that look relevant but should not take over the lesson.
