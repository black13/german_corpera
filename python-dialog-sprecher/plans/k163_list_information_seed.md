# K163 List Information Seed

## Purpose

K163 is hard to read as a raw `Kannbeschreibung` because it compresses several
teaching decisions into one sentence:

- the learner sees a short written German text
- the text is often list-like
- the topic is familiar
- the text is supported by basic vocabulary, international words, or visuals
- the learner passes one or more small facts to another person in the shared
  language

The teacher should not say: "Today we practice individual information." That is
not classroom language. The usable classroom version is:

`Read a tiny list. Find one useful fact. Tell another person.`

## Scene Ladder

- Big scene: everyday written information in public life
- Small scene: a menu, sign, schedule, poster, invitation, notice, brochure, or
  form-like list
- Roles: learner as helper, friend/colleague/guest/tourist as recipient, German
  text as source
- Speech act: select and pass on one useful fact
- Text or utterance: "It says ...", "The course starts ...", "Breakfast is ...",
  "The price is ..."
- Lexical field: depends on the text scene, such as food, time, transport,
  school, housing, hotel, shopping, office
- Grammar: chosen after the fact type is known

## What "Einzelne Informationen" Means In Practice

For teaching, "einzelne Informationen" should become visible data units:

- time: day, clock time, opening time, start time
- place: room, address, meeting point, direction
- price: cost, fee, ticket price
- person: name, contact, role
- object: item, food, drink, room, document
- instruction: do this, do not do this, bring this, show this
- simple status: open, closed, included, not included, available, not available

This is not a grammar topic by itself. It is a reading-and-mediation operation:
identify the useful unit and say it simply.

## Grammar Follows The Information Type

Do not start from an abstract grammar topic. Start from the fact the learner
must pass on.

- Time fact: `um`, `am`, `von ... bis`, numbers, days
- Place fact: `in`, `im`, `bei`, `links`, `rechts`, room/address words
- Price fact: `kosten`, `Euro`, numbers, "Das kostet ..."
- Person fact: `heißen`, `Name`, "Herr/Frau ...", "die Lehrerin", "der Arzt"
- Object fact: `ist`, `gibt es`, articles when they help, common nouns
- Instruction fact: imperatives or simple modal-like formulas such as "Bitte
  ...", "Sie müssen ...", "nicht ..."
- Inclusion fact: `mit`, `ohne`, `inklusive`, "Das ist dabei."

This is the main teacher-preparation rule for K163:

`information type -> utterance pattern -> lexical field -> grammar`

## Teacher Prompt Rules

- Begin with a concrete text scene, not with the abstract phrase "einzelne
  Informationen".
- Use one tiny written input at a time: menu line, timetable line, poster line,
  appointment note, price list, invitation line.
- Ask the student to find one fact first. Only later combine two facts.
- Keep the task as mediation: the student tells another person the useful fact
  in the shared language.
- If the student reads words but does not pass on a useful fact, redirect to the
  fact: "Was ist die Uhrzeit?" / "Was kostet es?" / "Wo ist es?"
- If the student gives a social closing formula, mark it as polite but not the
  target task.
- Accept broken A1 output if the fact is recoverable.

## Example Micro-Scenes

- Menu: "Frühstück 8 Euro, Kaffee inklusive" -> say the price and what is
  included.
- Timetable: "Bus Zentrum 14:30" -> say when the bus leaves.
- Notice: "Deutsch A1, Montag 18:00, Raum 2" -> say day, time, or room.
- Invitation: "Sommerfest, Freitag, Garten, 15 Uhr" -> say where or when it is.
- Hotel sign: "Frühstück von 7 bis 10 Uhr" -> say the breakfast time.
- Office note: "Anmeldung nur mit Pass" -> say what document is needed.

## Why The Teacher Matters

The raw `Kannbeschreibung` is a dense planning sentence. A learner may not be
able to read it, and the teacher should not simply repeat it.

The teacher's job is to make the hidden scene explicit:

`Here is a small written thing. You only need one useful fact. Tell your friend.`

Textbooks often show this through a scene page or exercise layout. This project
should make the scene logic explicit in the prompt and in the UI, so the human
reader can see why a menu line, bus line, invitation, or office note belongs to
the same K163 skill.

## Compact Prompt Shape

`K163 = short list-like text + familiar topic + one useful fact + tell another person`

`Information units = time / place / price / person / object / instruction / status`

`Grammar follows the unit: time -> um/am/von...bis; price -> kostet; place -> in/im/bei; instruction -> bitte/nicht/müssen`
