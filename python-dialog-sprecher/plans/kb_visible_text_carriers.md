# Visible Text Carriers In A1 Kannbeschreibungen

## Core claim

A1 is not organized only by vocabulary or grammar. It is organized by:

```text
input carrier -> operation -> recipient -> utterance frame
```

For many written A1 tasks, the most important distinction is not the topic word
itself. The important distinction is whether the input stays visible while the
learner thinks.

## Better terms

Do not reduce this to "static objects versus dynamic actions." The cleaner
planning categories are:

- stable input carriers: `Schild`, `Aufschrift`, `Aushang`, `Formular`,
  `Fahrplan`, `Informationstafel`
- ephemeral input carriers: `Durchsage`, `Ansage`, spoken question, spoken
  instruction
- operations: `verstehen`, `weitergeben`, `übersetzen`, `notieren`,
  `aufschreiben`, `mitteilen`
- recipient/output direction: for self, for another person, in the shared
  language, into German, into writing

This matters because an object such as `Aushang` is not one skill by itself.
It can support several different skills:

- `Aushang verstehen` = reception
- information from an `Aushang weitergeben` = mediation
- appointment from an `Aushang notieren` = written production
- `Aushang erklären` = mediation/explanation

## The Gegenstand bleibt principle

K173-K176 are the easiest form of mediation because the source text does not
disappear. A spoken sentence is gone after the learner hears it. A sign, label,
notice, board, timetable, or form stays there.

The learner can:

- look
- think
- look again
- point
- say one German word or one tiny German sentence

For K173-K176, this produces the core chain:

```text
foreign visible text
-> stable written carrier
-> learner reads one word
-> German-speaking tourist/guest needs help
-> learner says: Das bedeutet X.
```

## K173-K176 application

K173 is the broad pattern:

```text
simple foreign-language information
-> signs/inscriptions
-> German-speaking person
-> one-word German mediation
```

K174-K176 are concrete offshoots:

```text
K174: shop door sign -> "fermé" -> "geschlossen"
K175: restroom door sign -> "caballeros" -> "Herren"
K176: hotel information board -> "breakfast" -> "Frühstück"
```

The teacher should not drift into broad hotel, restaurant, or shop talk. The
target action is always:

```text
see/read the visible word -> give the meaning to the German-speaking person
```

Core utterances:

- `Das bedeutet ...`
- `Das heißt ...`
- `Auf Deutsch: ...`
- `Das ist ...`
- `Ist das klar?`
- `Verstehen Sie?`

## K163 connection

K163 gives the broader planning rule:

```text
information type -> utterance pattern -> lexical field -> grammar
```

For signs, labels, notices, boards, and lists, the information type usually
comes before the grammar topic.

Common information types:

- status: `geöffnet`, `geschlossen`, `frei`, `besetzt`
- place: `Eingang`, `Ausgang`, `Rezeption`, `Toilette`
- object/service: `Frühstück`, `Parkplatz`, `Gepäckaufbewahrung`
- instruction: `Rauchen verboten`, `Nur für Personal`, `Bitte nicht stören`
- time: `von sieben bis zehn`, `bis elf Uhr`
- price: `8 Euro`, `kostenlos`, `inklusive`

Then grammar stays small and useful:

- `Das bedeutet ...`
- `Das heißt ...`
- `Das ist ...`
- `Der/Die/Das ... ist hier/links/rechts.`
- `... ist von ... bis ...`
- `Sie müssen ...`

## A1 ceiling

The verb `übersetzen` can look easy because it resembles English "translate,"
but the operation is cognitively heavy unless the source is tiny.

The A1 ceiling is not simply the verb. The A1 ceiling is marked by the size,
stability, and support around the source:

- `einfach`
- `kurz`
- `einzelne Wörter`
- `Basiswortschatz`
- `Internationalismen`
- `visuelle Elemente`
- `vertraut`
- `langsam`
- `deutlich`

So at A1, `übersetzen` usually means word-level or phrase-level transfer, not
full sentence or paragraph translation.

## Carrier matrix

```text
carrier             channel   persistence     operation      A1 output
Schild              written   stays visible   verstehen      word/meaning
Aufschrift          written   stays visible   übersetzen     Das bedeutet ...
Aushang             written   stays visible   notieren       Termin/Uhrzeit
Formular            written   stays visible   ausfüllen      Name/Adresse
Fahrplan            written   stays visible   weitergeben    Uhrzeit/Gleis/Ziel
Informationstafel   written   stays visible   erklären       Das ist ...
Durchsage           spoken    disappears      verstehen      key fact only
Frage               spoken    disappears      weitergeben    simple reported fact
Anweisung           spoken    disappears      notieren       short action note
```

This matrix should become part of the teacher-preparation logic. It tells the
teacher how hard the task is before choosing grammar.

## Where `Ausdrücke` fits

`Ausdrücke` is a real CEFR/Profile Deutsch term, but it is not the same kind of
thing as `Aufschrift`, `Schild`, or `Aushang`.

`Ausdruck` / `Ausdrücke` means expression, phrase, or formulaic language chunk.
It names language material, not the physical carrier that holds the language.

Examples from the A1 descriptors:

- `einfache Wörter, alltägliche Ausdrücke und sehr einfache Strukturen`
- `vertraute alltägliche Ausdrücke und einfache Sätze`
- `einzelne Wörter und sehr einfache Ausdrücke`
- `Wörter und Ausdrücke in öffentlichen Aufschriften`
- `bekannte Wörter oder Ausdrücke ... weitergeben`

So the relationship is:

```text
carrier: Aufschrift / Schild / Aushang / Informationstafel
content: Wort / Ausdruck / Satz / Information
operation: verstehen / übersetzen / weitergeben / notieren
```

Important example:

```text
Wörter und Ausdrücke in öffentlichen Aufschriften verstehen
```

Here `öffentliche Aufschriften` are the visible static carrier. `Wörter und
Ausdrücke` are the readable content inside that carrier.

There is also `ausdrucken`, meaning "to print." That is a separate verb and
belongs to a technology/software instruction task, not to the sign/notice
carrier pattern.

## Product implication

The quick guides should expose this as a visible planning layer:

```text
carrier -> channel/persistence -> operation -> recipient -> utterance frame
```

For example, K176 should be shown not just as "hotel vocabulary" but as:

```text
Informationstafel
-> stable written carrier
-> read one simple foreign word
-> German-speaking guest needs help
-> say: Das bedeutet Frühstück.
```

The grader should then ask:

- Did the teacher keep the visible carrier in the scene?
- Did the student transfer the meaning of the visible item?
- Was the recipient correct: German-speaking tourist/guest/person?
- Was the answer A1-sized: one word, phrase, or tiny sentence?
- Did the conversation drift into general scene talk instead of mediation?

## Short reusable formula

```text
A1 written mediation = stable visible carrier + one useful item + recipient who needs it + tiny transfer phrase
```

For K173-K176:

```text
Aufschrift/Schild/Tafel bleibt.
Student liest ein Wort.
Deutschsprachige Person fragt.
Student sagt: Das bedeutet ...
```
