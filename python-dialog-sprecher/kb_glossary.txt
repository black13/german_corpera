KB Glossar — Wörter die nicht im A1 Wortschatz sind
=======================================================

  Kommunikation             = communication — speaking and listening together
  Begrüßung                 = greeting — saying Guten Tag, Hallo
  Verabschiedung            = farewell — saying Tschüss, Auf Wiedersehen
  Vorstellung               = introduction — sich vorstellen, telling who you are
  Befinden                  = wellbeing — how you feel, Wie geht's?
  Vorlieben                 = preferences — what you like, ich mag
  Abneigungen               = dislikes — what you don't like
  Bitte                     = request — asking for something, Können Sie...
  Auskunft                  = information — telling facts about yourself
  Grundlagen                = basics — fundamental A1 skills
  Rezeption                 = reception — understanding what you hear or read
  Produktion                = production — speaking or writing yourself
  Interaktion               = interaction — two-way communication
  Sprachmittlung            = mediation — passing info between languages
  mündlich                  = spoken — using ears and mouth
  schriftlich               = written — using eyes and hands
  Anweisung                 = instruction — someone tells you what to do
  Wegerklärung              = direction — someone says how to get there
  Durchsage                 = public announcement — voice over speaker
  Aufschrift                = inscription — words on a sign
  Schild                    = sign — board with words, like Eingang
  Fahrplan                  = timetable — train/bus schedule
  Formular                  = form — paper with empty fields
  Anleitung                 = instructions — written steps to follow
  Notiz                     = note — short written message
  Angaben                   = data/details — facts like name, address
  aufschreiben              = to write down — copy or note something
  notieren                  = to note — write a short fact
  weitergeben               = to pass on — tell someone what you read/heard
  bewältigen                = to manage/cope — get through a situation
  vertraut                  = familiar — something you know well
  alltäglich                = everyday — normal daily life
  elementar                 = elementary — the simplest level
  konkret                   = concrete — specific, not abstract
  geläufig                  = common — seen or heard often
  übersichtlich             = clear — easy to read, well-organized
  illustriert               = illustrated — with pictures
